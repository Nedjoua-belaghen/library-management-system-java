# Library Management System -Java
This is my second Java desktop application developed using NetBeans and Java swing. the application is designed to manage a library while 
applying Object-Oriented Programming (OOP) principles.
## Features
  - Add books
  - Delete books
  - Search books by name
  - Borrow books
  - Return borrowed books
  - Administrator login
  - Suggest new books
  - save library data to a text file
## Object-Oriented Programming Concepts Used
   - Classes and Objects
   - Encapsulation
   - Inheritance
   - Interfaces
   - Polymorphism
   - Constructors
   - Method Overriding
   - Separation between application logic and graphical interface
## Technologies Used
   - Java
   - Java Swing
   - NetBeans
   - ArrayList
   - File Handling
